# Lifynk---A-Verified-Blood-Coordination-Platform
## 🎯 Descripition

Lifynk is a centralized, real-time, verified blood coordination
web platform designed to digitalize and streamline the
complete blood donation and distribution workflow. The
platform connects five interconnected pillars — Donors,
NGOs, Blood Banks, Hospitals, and Recipients — through a
single unified digital ecosystem.
